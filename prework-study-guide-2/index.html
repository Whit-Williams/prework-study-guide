HTML


  • HTML provides the structure of your webpage
  • This is structured by two main sections: the head and the body.
  • The "head" section, provides the browser with everything it needs to render the web page corrently. This can include declarations, metadata, page titles and other helpful elements.
  • The "body" section is where the fun begins, adding elements that will be visible to the end user. This can include CSS and Javascript to further build on the HTML elements, but essentially, you are adding the elements you want to appear on the page at this point and positioning them on the page.
